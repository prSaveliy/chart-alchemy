════════════════════════════════════════════
CHART TYPE REFERENCE
════════════════════════════════════════════

CARTESIAN CHARTS
─────────────────
• LINE — series[].type = "line". Use "smooth": true for curved lines. Support "stack" for stacked areas.
• BAR — series[].type = "bar". For horizontal bars swap xAxis/yAxis types (xAxis: type "value", yAxis: type "category"). Support "stack" for stacked bars.
• SCATTER — series[].type = "scatter". Data points: [[x, y], ...]. Add "symbolSize" for bubble effect.
• BUBBLE — series[].type = "scatter". Data points: [[x, y, size], ...]. Use a symbolSize function to scale by the third value.

STATISTICAL CHARTS
───────────────────
• BOXPLOT — series[].type = "boxplot". Provide pre-calculated [min, Q1, median, Q3, max] arrays.
• CANDLESTICK — series[].type = "candlestick". Data per item: [open, close, low, high].
• HEATMAP — series[].type = "heatmap". Requires "visualMap" component. Data: [[x-index, y-index, value], ...].

RELATIONAL / PART-OF-WHOLE CHARTS
───────────────────────────────────
• PIE — series[].type = "pie". Data: [{name, value}, ...]. Set "radius": "60%". Add "label": { "show": true }.
• DOUGHNUT — Same as pie but "radius": ["40%", "70%"].
• ROSE (Nightingale) — Same as pie but add "roseType": "radius".
• FUNNEL — series[].type = "funnel". Data: [{name, value}, ...] ordered largest to smallest. Add "sort": "descending".
• GAUGE — series[].type = "gauge". Include "min", "max", "data": [{"value": N, "name": "label"}]. Include "progress": {"show": true}. Include "detail": {"valueAnimation": true, "formatter": "{value}%"}.

HIERARCHICAL CHARTS
────────────────────
• TREE — series[].type = "tree". Data is a nested object: { "name": "root", "children": [...] }. Add "orient": "LR" for left-to-right. Add "layout": "orthogonal".
• TREEMAP — series[].type = "treemap". Data: [{ "name", "value", "children": [...] }]. Set "roam": true.
• SUNBURST — series[].type = "sunburst". Data: [{ "name", "value", "children": [...] }]. Set "radius": ["15%", "90%"].

FLOW / NETWORK CHARTS
──────────────────────
• SANKEY — series[].type = "sankey". Include "nodes": [{"name"}], "links": [{"source","target","value"}]. Set "orient": "horizontal", "layout": "none".
• GRAPH (network) — series[].type = "graph". Include "nodes": [{"id","name","symbolSize","x","y"}], "links": [{"source","target"}]. Set "layout": "force", "roam": true, "force": {"repulsion": 100}.
